  <div className="sub-div"/>
  <p style={{fontSize:'0.88rem'}}>Each row uses <code>int % 2</code> where <code>int</code> is a 32-bit unsigned integer (range 0 to 4,294,967,295). For a modulus <code>m</code>, bias exists when 2³² is not evenly divisible by <code>m</code>.</p>
